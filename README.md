# GC VPS Bootstrap

A production-ready VPS bootstrap framework for Ubuntu servers.

## Features

- System updates and package management
- SSH hardening
- UFW firewall configuration
- Fail2Ban protection
- Docker installation
- Docker Compose installation
- Tailscale setup
- GitHub Actions self-hosted runner setup
- Monitoring stack support
  - Prometheus
  - Grafana
  - Loki
  - Node Exporter
- Automated backups
- Nginx reverse proxy support

## Supported OS

- Ubuntu 22.04+
- Ubuntu 24.04+

## Goal

Provision a secure and production-ready server in minutes.
